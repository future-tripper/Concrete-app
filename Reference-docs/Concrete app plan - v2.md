
# CONCRETE v2: Full Redesign Plan

## The Core Insight

AI can’t let anything be ordinary. It borrows gravitas from prestigious domains and applies it everywhere:

- A pasta recipe becomes “nutritional coherence architecture”
- Server maintenance becomes “ethical custodianship of distributed systems”
- A morning jog becomes “somatic optimization protocol”
- Reiki becomes “a technology for regulating the nervous system”
- Branding becomes “a moral act of designing behavior”

This is **borrowed gravitas**—prestige imported from other domains. It’s not inherently wrong. It can be creative, even illuminating. But it’s usually *unconscious*, and when there’s nothing concrete underneath, it’s hollow.

**The test:** Is there substance beneath the elevation? Or is it gravitas all the way down?

-----

## The Danger (When Borrowed Gravitas Fails)

1. **Displacement** — Elevated language REPLACES specifics instead of illuminating them
2. **Unfalsifiability** — Abstract claims can’t be tested or challenged
3. **Borrowed authority without borrowed rigor** — Gets the prestige of science without the discipline
4. **Erosion of trust** — Readers sense the gap between Important™ language and lack of substance
5. **Homogenization** — Everyone sounds the same; actual expertise becomes invisible

**When it works:**

- When elevation is followed by specifics
- When cross-domain metaphors genuinely teach something
- When it’s deliberate and playful, not default and earnest
- When you CAN speak concretely—you’re choosing elevation, not hiding behind it

## Altitude Levels

|Altitude         |Description                                                                                                     |
|-----------------|----------------------------------------------------------------------------------------------------------------|
|**GROUNDED**     |Post is already concrete. Specific details, examples, evidence. May have some elevated language but it’s earned.|
|**MODERATE**     |Mix of elevation and specifics. Some gravitas borrowing but anchored in places.                                 |
|**HIGH**         |Heavily borrowed gravitas. The elevation is doing most of the work.                                             |
|**STRATOSPHERIC**|Gravitas all the way down. Multiple types borrowed. Dense phrase clusters. Pure abstraction.                    |

## Grounding Check

**ANCHORED ✓**

> “You’re using elevated language, but it’s earned—there are specific examples, details, or evidence holding it up.”

**NOT ANCHORED ❌**

> “The elevated language isn’t connected to concrete details, examples, or evidence. It’s gravitas all the way down.”

**ALREADY GROUNDED ✓**

> “Your post is already specific. The reader can trace your claims to concrete details.”

## “Ground It” Prompt (Phase 2a)

```
ROLE:
You are rewriting this post to anchor its elevated language in concrete specifics.

THE ORIGINAL POST:
[user's post]

THE ANALYSIS:
[analysis JSON from phase 1]

YOUR TASK:
Rewrite the post preserving the author's intent and insights, but:

1. Replace abstract claims with specific examples, details, or evidence
2. Name things: tools, practices, people, timeframes, outcomes
3. Answer the grounding questions implicitly: Who specifically? What specifically? What would someone see or measure?
4. Keep the author's voice—don't flatten it, just anchor it
5. If there's a genuine insight buried under the gravitas, surface it in plain language

The result should pass the grounding check: a reader should be able to picture something concrete or trace claims to specifics.

Keep it the same approximate length as the original.

OUTPUT:
Just the rewritten post. No preamble, no explanation.
```

## “Own It” Prompt (Phase 2b)

```
ROLE:
You are rewriting this post to FULLY COMMIT to its borrowed gravitas. The goal is to push the elevation to its logical extreme—so far that it becomes art, satire, or manifesto.

THE ORIGINAL POST:
[user's post]

THE ANALYSIS:
[analysis JSON from phase 1]

YOUR TASK:
1. Identify what gravitas types the post is already borrowing
2. Amplify THOSE types to the maximum—don't add different types, intensify what's there
3. If it's borrowing from physics, go full quantum field theory
4. If it's borrowing from systems, go full enterprise architecture at civilizational scale
5. If it's borrowing from wisdom, go full mystical prophecy
6. If it's borrowing from action, go full transformation mythology

TONE:
- Deadpan serious. No winking. No "just kidding."
- The humor comes from the commitment, not from signaling that you're joking
- Write it like you absolutely mean it
- Think: TED talk that has ascended to a higher plane

The result should make the original look restrained by comparison.

Keep it the same approximate length as the original (or slightly longer for dramatic effect).

OUTPUT:
Just the rewritten post. No preamble, no explanation.
```

## “Elevate It” Prompt (for already-grounded posts)

```
ROLE:
This post is already grounded and specific. The user wants to see what it would sound like with borrowed gravitas added.

THE ORIGINAL POST:
[user's post]

THE ANALYSIS:
[analysis JSON showing it's grounded]

YOUR TASK:
Take this concrete, specific post and elevate it—add borrowed gravitas while keeping the original specifics underneath. The result should be HIGH altitude but still ANCHORED.

This is the "earned elevation" version: you get to sound important because you've also shown the work.

Layer in:
- Scientific gravitas (if the topic suits it)
- Systems gravitas (if there's structure to elevate)
- Wisdom gravitas (if there's insight to make profound)

But keep the original concrete details visible. The reader should see both the elevation AND what's underneath.

OUTPUT:
Just the rewritten post. No preamble, no explanation.
```

## UI States

### State 1: Empty

- Value prop
- How it works (collapsible)
- Textarea
- “Analyze” button (disabled)

### State 2: Loading

- “Analyzing…” on button
- Optional: fun loading states (“Measuring altitude…”, “Checking for ground…”)

### State 3: Analysis Complete

Show analysis card:

- Altitude badge (with icon/emoji)
- Altitude description (the colorful one-liner)
- Gravitas composition bars
- Grounding check (anchored or not, with explanation)
- Key phrases (collapsible, as supporting evidence)

Then show path buttons:

- If NOT ANCHORED: “Ground It” and “Own It”
- If ANCHORED: “You’re anchored ✓” and optional “Elevate Further”
- If GROUNDED: “Already grounded ✓” and “Take It Higher”

### State 4: Rewrite Complete

- Show the rewritten post
- Copy button
- “Try the other path” button
- “Start over” button

## Technical Implementation

### API Calls

- **Call 1:** Analyze → returns analysis JSON
- **Call 2:** Rewrite (with chosen path) → returns rewritten post

Two calls is better than one because:

1. Faster initial response
2. Only generate the rewrite they choose
3. Cleaner prompts

## Summary

CONCRETE v2 shifts from:

- ❌ “Vapor is bad, here are the bad words”
- ✓ “You’re borrowing gravitas—did you mean to? Is it anchored? Here are your options.”

The app becomes:

1. A diagnostic tool (what are you borrowing, is it anchored)
2. A creative tool (two valid paths: ground it or own it)
3. A rewriting tool (actual output you can use)

The tone shifts from finger-wagging to collaborative. The user has agency. Both paths are valid. The only wrong choice is unconscious borrowing that displaces substance.
